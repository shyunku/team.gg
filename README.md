## Notice

- 현재 버전은 베타 버전으로, 일부 기능이 동작하지 않을 수 있습니다.
- 현재 버전은 KR 서버를 제외한 다른 지역을 지원하지 않습니다.
- PR은 언제나 환영합니다. (단, Merge는 상황에 따라 지연되거나 Reject될 수 있습니다.)
